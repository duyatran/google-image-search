FreeCodeCamp API Basejump: URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage

https://obscure-oasis-78905.herokuapp.com/new/https://www.google.com https://obscure-oasis-78905.herokuapp.com/new/http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"https://obscure-oasis-78905.herokuapp.com/C" }

Usage

https://obscure-oasis-78905.herokuapp.com/C

Will redirect to:

http://foo.com:80
